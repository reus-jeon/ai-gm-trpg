# AI GM TRPG — 프로젝트 브리프

> 판타지 세계관에서 AI가 게임마스터(GM)를 맡아 진행하는 싱글플레이 TRPG 웹사이트
> Claude API 기반 · 세미클래스 바이브코딩 · 2026.06.20

---

## 인터뷰 응답 요약

| 항목 | 응답 |
|------|------|
| 플레이 방식 | AI가 GM · 멀티플레이 지향 (MVP는 싱글) |
| 세계관 | 판타지 |
| 세션 구조 | 방 만들기 / 입장 코드 참여 (오픈 세션) |
| 캐릭터 생성 | 직업·종족 선택 → AI가 주사위 굴려 능력치 결정 |
| 행동 입력 | 채팅 자유 텍스트 |
| 데이터 저장 | 세션 중에만 유지 (localStorage / 메모리) |
| 디자인 분위기 | 어두운 판타지 — 검정/금색, 양피지 느낌 |

---

## MVP 핵심 기능

| 기능 | 내용 |
|------|------|
| 방 생성 / 입장 | 방장이 방 만들기 → 코드 공유 → 다른 플레이어 입장 (멀티 확장 대비) |
| 캐릭터 생성 | 직업·종족 선택 → AI가 주사위 굴려 능력치 결정 |
| 채팅 기반 플레이 | 플레이어 자유 텍스트 행동 입력 |
| AI GM | Claude API — 상황 묘사·주사위 판정·이야기 진행 |
| 세션 상태 유지 | localStorage / 메모리 (로그인·DB 없음) |

---

## 디자인 방향

- 어두운 판타지 테마 (Dark Fantasy)
- 컬러: 검정 `#0D0A07` + 금색 `#C9A227` + 양피지 `#E8D5A3`
- 폰트: Cinzel (판타지 헤딩) + IM Fell English (본문 세리프)
- UI 요소: 양피지 텍스처, 테두리 장식, 주사위 결과 강조

---

## 기술 스택 (추천)

- Frontend: Next.js → Vercel 배포
- AI: Claude API (`claude-sonnet-4-6`)
- 실시간 멀티 (2단계): Supabase Realtime
- 저장: 세션 메모리만 (DB 없음 — MVP 기준)
